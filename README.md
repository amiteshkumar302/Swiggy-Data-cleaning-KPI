# Swiggy-Data-cleaning-KPI
Swiggy Data Analysis Project | SQL
Performed end-to-end data analysis on Swiggy food delivery data using SQL to extract actionable business insights.
Designed and executed complex SQL queries involving JOINs, subqueries, CTEs, GROUP BY, HAVING, and window functions.
Analyzed customer behavior, order trends, restaurant performance, delivery time, and revenue metrics.
Identified top-performing restaurants, peak order hours, repeat customers, and high-revenue locations to support data-driven decision-making.
Cleaned and transformed raw datasets by handling NULL values, duplicates, and data inconsistencies.
Optimized queries to improve performance and execution efficiency.
Generated insights useful for business growth, marketing strategy, and operational optimization.
Tools & Technologies: SQL, MySQL/PostgreSQL, Data Analysis, Relational Databases
